👨‍🎓 Students

What it does: Your complete student register — every learner enrolled in the school.

Who uses it: Admin & staff add/edit; parents see only their own children.

How to use it:

  1. Click + Add new to enroll a student (the Admission No is auto-generated — never type it).
  2. Pick the Class from the dropdown so the student is grouped correctly.
  3. Use Import CSV to register many students at once (download the template first).
  4. Every other page (results, fees, attendance…) pulls student names from here, so there is no re-typing.
Open Students

👨‍🏫 Staff

What it does: The directory of all teaching and non-teaching staff.

Who uses it: Admin/HR maintain it; approved staff sign-ups appear here automatically.

How to use it:

  1. Click + Add new; choose Teaching or Non-teaching.
  2. For teachers, pick the Subject taught from the dropdown.
  3. For privacy, date of birth is captured as day & month only.
  4. A Staff No is auto-generated on save.
Open Staff

📚 Classes

What it does: Defines each class/arm the school runs (e.g. JSS1 A).

Who uses it: Admin sets these up at the start of the session.

How to use it:

  1. Click + Add new; type the class name and arm.
  2. Pick the Class teacher from the staff dropdown (only teaching staff appear).
  3. Set the level and capacity.
  4. These classes then appear as dropdown options everywhere a class is needed.
Open Classes

◦ Subjects

What it does: The list of every subject the school offers.

Who uses it: Admin/HOD register subjects once per session.

How to use it:

  1. Click + Add new; type the subject, code, department and level.
  2. Map it to a Subject teacher from the staff dropdown.
  3. Subjects then appear in results, scheme of work, assignments and the timetable.
Open Subjects

📋 Attendance

Who uses it: Class teachers record it; parents view their own children.
